En este archivo se explicaran las funciones de la pagina.
Tenemos la pantalla principal a la cual se puede acceder mediante el texto que dice Django en el NAV

Productos:
Al ingresar a esta pantalla mediante el boton homónimo en el nav, se pediran 3 campos, NOMBRE, PRECIO Y MARCA para ingresar un nuevo
objeto a la base de datos, al presionar guardar, el mismo se almacena en la base.

Clientes:
Al ingresar a esta pantalla mediante el boton homónimo en el nav, se pediran 2 campos, NOMBRE Y EMAIL para ingresar un nuevo
objeto a la base de datos, al presionar guardar, el mismo se almacena en la base.

Empleados:
Al ingresar a esta pantalla mediante el boton homónimo en el nav, se pediran 3 campos, NOMBRE, PUESTO Y FECHA DE INGRESO
para ingresar un nuevo objeto a la base de datos, al presionar guardar, el mismo se almacena en la base

Buscar:
En esta pantalla se presentan 3 campos, uno de busqueda de producto, otro de cliente y el ultimo de empleado. Al ingresar el nombre del
objeto que queremos encontrar, se mostrara en otra pantalla los resultados de busqueda, hay un boton de busqueda independiente para 
cada campo.
